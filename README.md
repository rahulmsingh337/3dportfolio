# Rahul Singh — Portfolio

Dark portfolio built with **React + Vite + Tailwind CSS + GSAP + Framer Motion**.

## Stack
- React 18 + Vite
- Tailwind CSS v4 (@tailwindcss/vite)
- GSAP (ScrollTrigger, entrance animations)
- Framer Motion (AnimatePresence, loading screen)

## Features
- Custom animated loading screen
- GSAP hero entrance animations
- Scroll-driven reveal on all sections
- Custom cursor
- Animated marquee
- Fully responsive

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
